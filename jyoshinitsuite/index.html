# 前文

  • 恵まれた環境と恵まれた能力とを恵まれない人々を貶めるためにではなく、そういう人々を助けるために使ってください。

# 「は」と「が」の区別

# 「は」

  • 後ろの内容を強調する
私はXXXです。
  • 対比の意味を含める
お父さんは医者ですが、お母さんは弁護士です。
私は会社に行きましたが、弟はデパートに行きました。
猫が寝ています、猫は寝ています➡猫は寝ていますが、犬は起きています。

---
寿司は好きです。 　// 主語以外、否定される
（私は）寿司は好きです➡意味が二つあります。　// ①寿司が好きです。②　寿司以外は嫌いです。
（私は）寿司は好きですが、寿司以外は嫌いです。　// ②のニュアンスが入るか入らないかは文脈で変わります。

今日はきれいですね。　// 今日だけ、きれいですが、この前は全部きれいではなかった。

今日もきれいですね。
                  // 今日は、よりきれいですね。今日は、いつにも増してきれいですね。「いつもきれいだけど、今日は一段ときれい」の意味を含める
いつもきれいですね。
// 助詞を抜かすやり方もあります。　=> 今日、きれいですね。

# 「が」

  • 前の内容を強調する
XXXさんがいらっしゃますか
- 私がXXXです。
  • （自然）現象とか、目に見える物の「状況」「状態」「存在」 とかを表す
風が吹いています（状況）
猫が寝ています（状態）
赤い車があります（存在）
  • 背景を説明する
むかしむかしあるところにおじいさんとおばあさんが住んでいました。
