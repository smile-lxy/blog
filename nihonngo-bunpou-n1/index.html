# 〜済むことではない[1]

# 意味
  • 不是光… 能解决的
# 接続
動て型
名詞＋で　＋済むことではない
発話＋で
# 例文
何でも謝って済むことではないけれど、謝れない人間は最低だ。
それは謝罪と賠償だけで済む問題ではないと思う。
一人が責任を取って辞めて済むことではない。

# 〜ともなると・ともなれば[2]

# 意味
  • 一到… 的时候
# 接続
動辞型
名詞＋で　＋ともなると・ともなれば
# 例文
この年齢ともなると体が思うように動いてくれなくなる。
１２月ともなれば冷え込んでくる。
