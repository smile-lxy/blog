# 前言

# 动词分类

# 介绍
  • 动词是表示主体 (人或事物) 的存在，行为，动作或状态的词，日语的动词分为词干和词尾部分，词尾的变化称之为活用
  • 动词按其形态和活用规律可分为：五段活用动词，一段活用动词，サ行变格活用动词，カ行变格活用动词。动词的原形叫 ' 基本形'
# 规则
サ变动词: 「する」, 指定性名词后接する构成的动词 例: 勉強する・練習する　など
カ变动词: 「来る（くる）」, 初级只有这个是カ变动词
一段动词: 词尾是「る」, 且倒数第二个假名是以「イ・エ」段结尾的动词 例: 寝る・食べる　など
五段动词: 词尾不是「る」, 五段动词; 词尾是「る」, 且倒数第二个假名是以「ア・ウ・オ」结尾的动词 例: 洗う・作る　など
特殊五段: 词尾是「る」, 倒数第二个假名是以「イ・エ」段假名结尾 例: 帰る(かえる)・走る(はしる)・入る(はいる)・知る(しる)・滑る(すべる)・切る(きる)　など

判断技巧:
1.是不是サ变, カ变, 特殊五段动词
2.词尾是不是る:
      不是: 五段
        是: 倒数第二个假名是不是以い段或え段假名结尾:
             是: 一段
           不是: 五段

# 正文

# 原 (辞书，基本，连体) 形

  • 后续必须接名，体言，数量或代词

# 简体 (普通) 形

  • 用于关系亲密的人之间，动词是用原形 (基本形・辞书形)
    • 行く

# マス (连用) 形

  • 用于对长辈，　上级或不熟悉的人，　表示礼貌.　动词　＋　ます
# 一段 (二类) 动词
  • 去掉る
    • 食べる (たべる) -> 食べます
    • 教える (おしえる) -> 教えます
    • 入れる (いれる) -> 入れます
# 五段 (一类) 动词
  • 词尾假名 -> 该行イ段假名
    • 書く (かく) -> 書き
# サ变 (三类) 动词
  • る -> し
    • 勉強する -> 勉強します
# カ变 (三类) 动词
  • 来る（くる）-> 来 (き)
    • 来る (くる) -> 来ます (きます)

＊ 接在「マス」形的特殊动词

なさるなさいます
いらっしゃるいらっしゃいます
おっしゃるおっしゃいます
ござるございます

# テ形

# 一段 (二类) 动词
  • 去掉る
    • 食べる (たべる) -> 食べて
    • 教える (おしえる) -> 教えて
    • 入れる (いれる) -> 入れて
# 五段 (一类) 动词
  • 以 ' う　つ　る’结尾，要发生促音变
    • 買う (かう) -> 買って
    • 待つ (まつ) -> 待って
    • 帰る (かえる) -> 帰って
  • 以‘く’或 ' ぐ' 结尾，-> ‘い’
    • 書く (かく) -> 書いて
    • 急ぐ (いそぐ) -> 急いで
  • 以‘ぶ　む　ぬ’结尾，-> ‘ん’
    • 遊ぶ (あそぶ) -> 游んで
    • 読む (よむ) -> 読んで
    • 死ぬ (しぬ) -> 死んで
  • 以‘す’结尾，-> ‘し’
    • 話す (はなす) -> 話して
# サ变 (三类) 动词
  • する -> し
    • 勉強する -> 勉強して
# カ变 (三类) 动词
  • 来る（くる）-> 来 (き)
    • 来る (くる) -> 来て (きて)
# イ (一类) 形容词
  • い -> く
    • 美味しい (おいしい) -> 美味しくて
# ナ (二类) 形容词，名词
  • 词尾 + で
    • 綺麗 (きれい) -> 綺麗で
    • 学生 (がくせい) -> 学生で

# タ形 (变形同て形一样)

# 一段 (二类) 动词
  • 去掉る
    • 食べる (たべる) -> 食べた
    • 教える (おしえる) -> 教えた
    • 入れる (いれる) -> 入れた
# 五段 (一类) 动词
  • 以 ' う　つ　る’结尾，要发生促音变
    • 買う (かう) -> 買った
    • 待つ (まつ) -> 待った
    • 帰る (かえる) -> 帰った
  • 以‘く　ぐ ' 结尾，-> ‘い’
    • 書く (かく) -> 書いた
    • 急ぐ (いそぐ) -> 急いだ
  • 以‘ぶ　む　ぬ’结尾，-> ‘ん’
    • 遊ぶ (あそぶ) -> 游んだ
    • 読む (よむ) -> 読んだ
    • 死ぬ (しぬ) -> 死んだ
  • 以‘す’结尾，-> ‘し’
    • 話す (はなす) -> 話した
# サ变 (三类) 动词
  • する -> し
    • 勉強する -> 勉強した
# イ (一类) 形容词
  • い -> かった
    • 美味しい (おいしい) -> 美味しかった
# ナ (二类) 形容词
  • 词尾 + だった
    • 綺麗 (きれい) -> 綺麗だった

# ナイ (未然) 形

# 一段 (二类) 动词
  • 去掉る
    • 食べる (たべる) -> 食べない
    • 教える (おしえる) -> 教えない
    • 入れる (いれる) -> 入れない
# 五段 (一类) 动词
  • 词尾假名 -> 该行ア段假名
    • 書く (かく) -> 書かない
  • 以 ' う' 结尾 -> わ
    • 買う (かう) -> 買わない
# サ变 (三类) 动词
  • する -> し
    • 勉強する -> 勉強しない
# カ变 (三类) 动词
  • 来る（くる）-> 来 (こ)
    • 来る (くる) -> 来ない (こない)

# バ (假定) 形

# 一段 (二类) 动词
  • る -> れ
    • 食べる (たべる) -> 食べれば
    • 教える (おしえる) -> 教えれば
    • 入れる (いれる) -> 入れれば
# 五段 (一类) 动词
  • 词尾假名 -> 该行エ段假名
    • 書く (かく) -> 書けば
# サ变 (三类) 动词
  • る -> れ
    • 勉強する -> 勉強すれば
# カ变 (三类) 动词
  • 来る（くる）-> 来れ (くれ)
    • 来る (くる) -> 来れば (くれば)
# イ (一类) 形容词
  • い -> けれ
    • 美味しい (おいしい) -> 美味しければ
# ナ (二类) 形容词，名词
  • 词尾 + なら
    • 綺麗 (きれい) -> 綺麗なら
    • 学生 (がくせい) -> 学生なら

# 命令・禁止形

# 一段 (二类) 动词
  • る -> ろ
    • 食べる (たべる) -> 食べろ
    • 教える (おしえる) -> 教えろ
    • 入れる (いれる) -> 入れろ
# 五段 (一类) 动词
  • 词尾假名 -> 该行エ段假名
    • 書く (かく) -> 書け
# サ变 (三类) 动词
  • る -> ろ
    • 勉強する -> 勉強しろ，勉強せよ
# カ变 (三类) 动词
  • 来る（くる）-> 来い (こい)

# 意向形

# 一段 (二类) 动词
  • 去掉る + よう
    • 食べる (たべる) -> 食べよう
    • 教える (おしえる) -> 教えよう
    • 入れる (いれる) -> 入れよう
# 五段 (一类) 动词
  • 词尾假名 -> 该行ウ段假名 + う
    • 書く (かく) -> 書こう
# サ变 (三类) 动词
  • する -> しよう
    • 勉強する -> 勉強しよう
# カ变 (三类) 动词
  • 来る（くる）-> 来よう (こよう)

# 可能态

# 一段 (二类) 动词
  • 去掉る + られる
    • 食べる (たべる) -> 食べられる
    • 教える (おしえる) -> 教えられる
    • 入れる (いれる) -> 入れられる
# 五段 (一类) 动词
  • 词尾假名 -> 该行エ段假名 + る
    • 書く (かく) -> 書ける
# サ变 (三类) 动词
  • する -> できる
    • 勉強する -> 勉強できる
# カ变 (三类) 动词
  • 来る（くる）-> 来られる (こられる)

# 被动态

# 一段 (二类) 动词
  • 去掉る + られる
    • 食べる (たべる) -> 食べられる
    • 教える (おしえる) -> 教えられる
    • 入れる (いれる) -> 入れられる
# 五段 (一类) 动词
  • 词尾假名 -> 该行ア段假名 + れる
    • 書く (かく) -> 書かれる
# サ变 (三类) 动词
  • する -> される
    • 勉強する -> 勉強される
# カ变 (三类) 动词
  • 来る（くる）-> 来られる (こられる)

# 使役形

# 一段 (二类) 动词
  • 去掉る + させる
    • 食べる (たべる) -> 食べさせる
    • 教える (おしえる) -> 教えさせる
    • 入れる (いれる) -> 入れさせる
# 五段 (一类) 动词
  • 词尾假名 -> 该行ア段假名 + せる
    • 書く (かく) -> 書かせる
# サ变 (三类) 动词
  • する -> させる
    • 勉強する -> 勉強させる
# カ变 (三类) 动词
  • 来る（くる）-> 来させる (こさせる)

# 使役被动形

# 一段 (二类) 动词
  • 去掉る + させられる
    • 食べる (たべる) -> 食べさせられる
    • 教える (おしえる) -> 教えさせられる
    • 入れる (いれる) -> 入れさせられる
# 五段 (一类) 动词
  • 词尾假名 -> 该行ア段假名 + せられる
    • 書く (かく) -> 書かせられる
# サ变 (三类) 动词
  • する -> させられり
    • 勉強する -> 勉強させられる
# カ变 (三类) 动词
  • 来る（くる）-> 来させれらる (こさせられる)

# 形容词

  • 形容词是表示事物性质和状态的词，都以「い」结尾，形容词的词尾可用变化，这种变化称之为 ' 活用'
# 谓语
  • 形容词作谓语时的形态叫 ' 终止形 ', 终止形后续「です」构成敬体语
    • この店の料理は美味しいです。(这家店的菜很好吃)
# 定语
  • 形容词直接修饰名词作定语，修饰名词时的形态叫做 ' 连体形 '(形容词的连体形和终止形一样).
    • この高い建物は図書館です。(这个高的建筑物是图书馆)

# 形容动词

  • 形容动词是表示事物性质和状态的词，词尾为「だ」, 但一般不写出来
# 谓语
  • 形容动词作谓语时，「だ」->「です」
    • この町は静かですね。
# 定语
  • 形容动词作定语时，「だ」->「な」, 修饰名词时的形态叫做 ' 连体形'
    • この綺麗な花はなんの花ですか。
