# 国

# 研修中

  • 新規就農者育成総合対策 (就農準備資金)
    • 150 万円 / 年　(最長２年)

# 開始

  • 新規就農者育成総合対策 (経営開始資金)
    • 150 万円 / 年　(最長３・５年)
  • 青年等就農資金
    • 最大 3,700 万円 (無利子)
  • 経営体育成強化資金
    • 最大 500 万円融資 (農地購入費)
  • 新規就農者育成総合対策 (経営発展支援事業)
    • 3 の 4 支援 (機械・施設等 (上限あり))

# 市

# 研修中

  • 住宅家賃助成
    • 5,000・1 万円 / 月

# 開始

  • 就農支援資金
    • 200 万円 (農外者)
