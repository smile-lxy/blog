# 挣值管理 (EVM)

  • BAC (完工预算)
  • AC (实际成本)
  • EV: 挣值
  • PV: 计划价值
  • SV (进度偏差): EV - PV
  • CV (成本偏差): EV - AC
  • SPI (进度绩效指数): EV / PV
  • CPI (成本绩效指数): EV / AC
  • ETC (完工尚需估算):
  • EAC (完工估算): AC + ETC
    • 非典型 (剩余工作按照计划的绩效继续): AC + (BAC - EV) / 1 = AC + BAC - EV
    • 典型 (剩余工作按照当前的绩效继续): AC + (BAC - EV) / CPI = BAC / CPI
    • 特殊绩效 (剩余工作同时受 CPI 和 SPI 的影响): AC + (BAC - EV) / (CPI * SPI)
    • 最准确: AC + 自下而上重新估算的 ETC
  • TCPI (完工尚需绩效指数) :
    • BAC 可行情况下: (BAC - EV) / (BAC - AC)
    • BAC 不可行情况下: (BAC - EV) / (EAC - AC)
  • VAC (完工偏差) : BAC - EAC
  • ES: 挣得进度
  • AT: 实际时间
  • TV (时间偏差): ES - AT
  • TPI (是时间绩效指数): ES / AT
  • SV > 0 | SPI > 1: 进度提前
  • SV = 0 | SPI = 1: 进度符合
  • SV < 0 | SPI < 1: 进度落后
