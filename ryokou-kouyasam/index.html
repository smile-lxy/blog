# 例外

  • 道に不慣れで、少し時間を無駄にしてしまった。
  • 車の左前のタイヤがパンクしてしまい、午後のほとんどの時間を費やすことになった。

# 助かり

  • タイヤがパンクした際、渓先生にはレンタカー会社や JAF のスタッフの方にご連絡いただき、本当に助かった。

# 教訓

  • 可能であれば、予算 (観光地の入場料や駐車料金など) を事前に計算しておくと良い。
  • 当日スムーズに選択できるよう、旅行先周辺の食事スポットを事前に調べておく。
  • マップでルートを調べる際には、目的地の駐車場を調べるべきである（プランを立てるときに調べておくと、より良い）。
  • 高速道路を利用する際は、事前に小銭や ETC カードを準備しておく。
  • 道に不慣れであるリスクを考慮に入れ、計画を立てたほうがよい。
  • 車の故障リスクも考慮し、計画に組み込むとともに、応急処置の方法も準備しておくのが望ましい。
  • 運転技術を向上させ、簡単な修理技術を身につけておくことも必要である。
  • 助けを頼む必要がある場合、頼れる人に事前に連絡を取っておくと良い。
