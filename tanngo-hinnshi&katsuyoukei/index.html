# 普通形（（ふつうけい）

  • 辞書形（じしょけい）
  • 現在 (げんざい）
  • 過去（かこ）
  • 肯定形（こうていけい）
  • 否定形（ひていけい）
品詞辞書形現在 - 肯定形現在 - 否定形過去 - 肯定形過去 - 否定形
動詞食べる食べる食べない食べた食べなかった
〜行く行く行かない行った行かなかった
〜勉強する勉強する勉強しない勉強した勉強しなかった
〜来る（くる）来る（くる）来ない（こない）来た（きた）来なかった（こなかった）
形容詞美しい美しい美しくない美しかった美しくなかった
形容動詞綺麗綺麗だ綺麗ではない・綺麗じゃない綺麗だった綺麗ではなかった・綺麗じゃなかった
名詞紅葉紅葉だ紅葉ではない・紅葉じゃない紅葉だった紅葉ではなかった・紅葉じゃなかった

# 丁寧形（ていねいけい）

品詞辞書形現在 - 肯定形現在 - 否定形過去 - 肯定形過去 - 否定形
動詞食べる食べます食べません食べました食べませんでした
〜行く行きます行きません行きました行きませんでした
〜勉強する勉強します勉強しません勉強しました勉強しませんでした
〜来る（くる）来ます（きます）来ません（きません）来ました（きました）来ませんでした（きませんでした）
形容詞美しい美しいです美しくないです・美しくありません美しかったです美しくなかったです・う美しくありませんでした
形容動詞綺麗綺麗です綺麗ではありません・綺麗じゃないです綺麗でした綺麗ではなかったです・綺麗じゃありませんでした
名詞紅葉紅葉です紅葉ではありません・紅葉じゃないです紅葉でした紅葉ではなかったです・紅葉じゃありませんでした
